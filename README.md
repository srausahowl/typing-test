# Typing Speed Test

Measure your typing speed with live WPM, accuracy, and a timer. Character-level correct/wrong highlighting.

## Live Demo

https://srausahowl.github.io/typing-test/

## Usage

Open `index.html` in any modern browser. Zero dependencies, zero build step.

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Tech

- Vanilla HTML / CSS / JavaScript
- No frameworks, no build tools
